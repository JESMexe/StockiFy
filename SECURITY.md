# Security Policy

## Supported Versions

StockiFy es un proyecto en desarrollo activo y no mantiene versiones públicas
estables con soporte extendido.

Las actualizaciones de seguridad se aplican únicamente a la versión más reciente
del repositorio.

| Version | Supported |
|--------|-----------|
| Latest | ✅ |
| Older versions | ❌ |

---

## Reporting a Vulnerability

Si encontrás una vulnerabilidad de seguridad en este proyecto, te pido que **no la publiques directamente en issues públicos del repositorio**.

En su lugar, podés reportarla enviando un correo a:

📧 **[tu email]**

Por favor incluí:

- una descripción clara del problema
- pasos para reproducirlo
- el impacto potencial de la vulnerabilidad
- si es posible, una propuesta de solución

Voy a intentar revisar los reportes lo antes posible.

---

## Alcance de seguridad

Este proyecto fue desarrollado principalmente con fines educativos y de portfolio.
Aunque intento aplicar buenas prácticas de seguridad, **no se garantiza que el
software esté listo para uso productivo sin auditorías adicionales**.

Las áreas donde se presta especial atención incluyen:

- autenticación de usuarios
- validación de inputs
- manejo de sesiones
- acceso a base de datos
- envío de correos
- integración con servicios externos (OAuth)

---

## Uso responsable

Si descubrís una vulnerabilidad, agradezco que se reporte de forma responsable
para poder corregirla antes de que sea divulgada públicamente.

La colaboración responsable ayuda a mejorar la seguridad del proyecto.
